1. 사용자 권한 컨트롤을 1단계(바탕 화면을 흐리게 표시하지 않음) 이하로 설정하십시오.

2. 설정→게임에서 Xbox Game Bar를 비활성화하십시오.

3. 크롬이나 크로미엄, 일렉트론 기반 프로그램이 정상적으로 실행되지 않을 때에는 바로가기 속성의 대상 입력창 뒤에 --disable-gpu를 붙인 후 실행해 보십시오.

4. 게임이 목적인 경우, 게임을 DWM 종료 전에 미리 실행시켜놓으십시오. DWM 종료 후에는 게임이 정상적으로 실행되지 않을 수 있습니다.

5. DWM 없는 환경은 실사용 용도로 적합하지 않습니다. 게임이나 원하는 작업이 끝나면 DWM을 복구하는 것을 권장드립니다.

6. 바탕 화면을 꽉 채우는 검은 창이 뜨는 경우 해당 창의 '최소화'에 해당하는 위치를 마우스로 클릭하십시오. 마우스를 올리고 있을때 '최소화'라고 뜨는 부분을 클릭하시면 됩니다.


 *DWM 없는 환경에서는 다음과 같은 문제가 발생할 수 있습니다:

1. Ctrl+Alt+Del, 잠금, 로그아웃, 시스템 종료, 절전, UAC 보안 데스크톱 등이 동작하지 않습니다.

2. 설정, 계산기, Edge을 포함한 UWP 앱과 네트워크 설정 등의 일부 구성 요소를 사용할 수 없습니다.

3. 일부 프로그램(인스톨러, 자동 압축 해제, 플러그인을 통해 실행하는 게임 등)이 실행되지 않습니다.

4. 프로그램이 정상적으로 종료되지 않는 경우가 있습니다. 이 경우 해당 프로그램은 시스템 자원을 계속 사용하게 되며, 동시 실행이 불가능한 프로그램의 경우 새로 실행이 불가능할 수도 있습니다. 작업 관리자나 taskkill 명령어를 통해 해당 프로세스를 강제 종료하거나 DWM을 복구하여 해결할 수 있습니다.  

5. 프리징이 자주 발생합니다. DWM을 복구하면 대부분은 바로 해결됩니다.

 *스크립트를 관리자 권한으로 실행하시면 이 안내문을 건너뛸 수 있습니다.
 
 *강제 고전테마를 적용한 상태에서 DWM을 복구하면 그래픽 드라이버 오류가 발생하여 드라이버가 비활성화되며 로그아웃됩니다. 이를 방지하기 위해 DWM 복구 전 고전테마를 해제해 주십시오. 이러한 상황이 발생하였을 경우 재부팅하거나 장치 관리자에서 그래픽 드라이버를 사용 안함 후 사용해주시면 됩니다.
